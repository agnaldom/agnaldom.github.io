Docker

Docker é uma plataforma aberta para desenvolvedores e administradores de sistemas para criar, enviar e executar aplicativos distribuídos, seja em laptops, VMs de centro de dados ou na nuvem.

Nginx

Nginx [engine x] é um servidor HTTP e um proxy reverso, um servidor proxy de correio e um servidor proxy genérico TCP / UDP.

Apache 2

O Apche HTTP Server Project é um esforço para desenvolver e manter um servidor HTTP de código aberto para sistemas operacionais modernos, incluindo UNIX/Linux e Windows

Gitlab

Gitlab é o principal produto integrado para o desenvolvimento de software moderno. Conexão de gerenciamento de problemas, controle de versão, revisão de código, CI, CD e monitoramento.

MariaDB

LDAP

Samba

Samba é um software executado em servidores, responsável por estabelecer interações com redes constituídas por computadores operados em Windows.
